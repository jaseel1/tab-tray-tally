# Getting Started Guide

Welcome to the Restaurant POS System! This guide will help you get started with the system quickly.

## First Time Setup

### 1. Access the System
- Open your web browser and navigate to the POS system URL
- You'll see the login screen with two options: POS Login and Admin Login

### 2. Login Types

#### POS Login (For Staff)
- **Username**: Your assigned POS account name
- **Password**: Your assigned password
- **Access Level**: Billing, order history, basic menu viewing

#### Admin Login (For Managers)
- **Username**: Your admin account name  
- **Password**: Your admin password
- **Access Level**: Full system access including menu management and reports

### 3. First Login
1. Select your login type (POS or Admin)
2. Enter your credentials
3. Click "Login"
4. The system will load your account data automatically

## Main Interface Overview

### Navigation Tabs
- **Home**: Dashboard with quick stats and recent orders
- **Billing**: Main POS interface for taking orders
- **Reports**: Sales analytics and data export
- **Menu**: View and manage menu items (Admin only)
- **Orders**: Order history and search
- **Settings**: System configuration (Admin only)

### Home Dashboard
- **Daily Sales**: Today's total revenue
- **Orders Count**: Number of orders today
- **Popular Items**: Best-selling menu items
- **Recent Orders**: Latest transactions

## Taking Your First Order

### Step 1: Navigate to Billing
1. Click the "Billing" tab
2. You'll see the menu items on the left and cart on the right

### Step 2: Add Items to Cart
1. **Browse Categories**: Use category filter if available
2. **Search Items**: Use the search bar to find specific items
3. **Add to Cart**: Click on menu items to add them
4. **Adjust Quantity**: Use +/- buttons in the cart

### Step 3: Complete the Order
1. **Review Cart**: Check items and quantities
2. **View Total**: GST is calculated automatically
3. **Process Order**: Click "Process Order"
4. **Print Receipt**: Use "Print Receipt" button

## Key Features for Daily Use

### Quick Search
- Use the search bar in billing to quickly find items
- Search works on item names and categories

### Cart Management
- **Add Items**: Click menu items or use + button
- **Remove Items**: Use - button or trash icon
- **Clear Cart**: "Clear Cart" button removes all items
- **Quantity**: Adjust individual item quantities

### Order History
- View all past orders in the "Orders" tab
- Search by order ID, date, or amount
- Export order data (Admin only)

### Receipt Handling
- Preview receipts before printing
- Print receipts directly from the system
- Receipts include GST breakdown and order details

## Basic Troubleshooting

### Login Issues
- **Check Credentials**: Ensure username/password are correct
- **Contact Admin**: If credentials don't work, contact your administrator
- **Clear Browser Cache**: Sometimes helps with login issues

### Menu Not Loading
- **Check Internet**: Ensure stable internet connection
- **Refresh Page**: Try refreshing the browser
- **Contact Admin**: Report persistent issues

### Receipt Not Printing
- **Check Printer**: Ensure printer is connected and has paper
- **Browser Settings**: Allow pop-ups for receipt printing
- **Try Preview**: Use receipt preview if printing fails

## Best Practices

### Daily Operations
1. **Start of Day**: Login and verify menu items are current
2. **Regular Sync**: System syncs automatically, but check connection periodically
3. **End of Day**: Review daily sales in reports section

### Order Management
1. **Double Check**: Always verify order details before processing
2. **Customer Receipt**: Offer printed receipts to customers
3. **Order History**: Use order history to track and reference past orders

### Security
1. **Logout**: Always logout when leaving the system unattended
2. **Password**: Don't share your login credentials
3. **Report Issues**: Immediately report any system issues
