# POS Operations Guide

This guide covers all daily point-of-sale operations for restaurant staff.

## Quick Start Checklist

### Beginning of Shift
- [ ] Login to your POS account
- [ ] Verify menu items are current and available
- [ ] Check that receipt printer is working
- [ ] Review any special offers or unavailable items
- [ ] Test a sample transaction if needed

### During Shift
- [ ] Process orders efficiently
- [ ] Provide receipts to customers
- [ ] Handle cash and card payments
- [ ] Note any menu item issues
- [ ] Keep workspace organized

### End of Shift
- [ ] Complete all pending orders
- [ ] Review daily sales summary
- [ ] Report any system issues
- [ ] Logout securely

## Taking Orders

### Step-by-Step Order Process

#### 1. Access the Billing Screen
- Click the **"Billing"** tab in the main navigation
- The screen shows menu items on the left, cart on the right

#### 2. Find Menu Items
**Browse by Category:**
- Use category filters if available
- Categories appear as tabs or dropdown
- Click category to filter items

**Search Function:**
- Use the search bar at the top
- Type item name or partial name
- Results update in real-time

**Manual Browse:**
- Scroll through all available items
- Items show name, price, and image (if available)

#### 3. Add Items to Cart
**Single Item:**
- Click on the menu item card
- Item appears in cart with quantity 1

**Multiple Quantities:**
- Click item multiple times, OR
- Use +/- buttons in the cart
- Quantity updates automatically

**Item Details:**
- Each cart item shows:
  - Item name
  - Unit price
  - Quantity
  - Line total (price Ã— quantity)

#### 4. Review Order
**Cart Summary:**
- **Subtotal**: Pre-tax amount
- **GST**: Tax amount (configurable rate)
- **Total**: Final amount including tax

**Modify Order:**
- Increase quantity: Click **+** button
- Decrease quantity: Click **-** button
- Remove item: Click **trash** icon or reduce to 0
- Clear entire cart: Click **"Clear Cart"** button

#### 5. Process Payment
- Click **"Process Order"** button
- Order is immediately saved to the system
- Receipt preview appears automatically

#### 6. Complete Transaction
- Review receipt details
- Click **"Print Receipt"** to provide customer copy
- Cart automatically clears for next order

## Advanced Order Features

### Customer Information
Some orders may require customer details:
- Customer name (for order tracking)
- Phone number (for delivery/pickup)
- Special instructions or notes

*Note: Customer information fields may not be visible in current interface but can be added via order notes.*

### Payment Methods
Current system supports:
- **Cash** (default)
- Additional payment methods may be configured by admin

### Order Modifications
**Before Processing:**
- Items can be freely added/removed
- Quantities can be adjusted
- Cart can be completely cleared

**After Processing:**
- Orders cannot be modified
- New order must be created for additional items
- Contact admin for order corrections

## Menu Navigation Tips

### Efficient Item Finding
**For Popular Items:**
- Memorize locations of frequently ordered items
- Use search for items you know by name

**For New Items:**
- Browse categories systematically
- Ask customer for clarification on unfamiliar requests

**For Seasonal/Special Items:**
- Check with management about availability
- Note items that are temporarily unavailable

### Handling Unavailable Items
**If Item Not Found:**
1. Use search function to double-check
2. Check similar items in same category
3. Ask customer for alternative
4. Contact supervisor if item should be available

**If Item Exists But Unavailable:**
1. Inform customer of unavailability
2. Suggest similar alternatives
3. Note issue for management
4. Continue with available items

## Receipt Management

### Receipt Preview
After processing an order:
- Receipt preview appears automatically
- Shows all order details:
  - Order number
  - Date and time
  - Itemized list with quantities
  - Subtotal, GST, and total
  - Restaurant information

### Printing Receipts
**Standard Process:**
1. Click **"Print Receipt"** button
2. Browser print dialog appears
3. Select appropriate printer
4. Confirm print settings
5. Print receipt

**Troubleshooting Print Issues:**
- Check printer connection and paper
- Ensure browser allows pop-ups
- Try print preview if direct printing fails
- Contact IT support for persistent issues

### Customer Receipt Policy
- Always offer printed receipts
- Required for business records
- Customer may decline but offer anyway
- Keep copy for daily reconciliation

## Order History and Tracking

### Accessing Order History
- Click **"Orders"** tab in main navigation
- Shows chronological list of all orders
- Most recent orders appear first

### Order Details
Each order entry shows:
- **Order Number**: Unique identifier (e.g., ORD000001)
- **Date/Time**: When order was processed
- **Total Amount**: Final order value
- **Item Count**: Number of different items

### Searching Orders
**By Order Number:**
- Use search box to find specific order
- Enter full or partial order number

**By Date:**
- Orders are sorted by date/time
- Scroll to find orders from specific time period

**By Amount:**
- Search function may work with amounts
- Useful for finding large or specific-value orders

### Order Information Uses
- **Customer Inquiries**: Look up past orders
- **Reorders**: Repeat previous orders
- **Daily Reconciliation**: Verify sales totals
- **Issue Resolution**: Reference problem orders

## Handling Special Situations

### System Slowness
**If System is Slow:**
1. Be patient - don't click multiple times
2. Wait for operations to complete
3. Check internet connection
4. Refresh browser if necessary
5. Contact IT support if persistent

### Login Issues
**If Login Fails:**
1. Verify username and password
2. Check caps lock and typing
3. Contact administrator for password reset
4. Use backup processes if available

### Order Processing Errors
**If Order Won't Process:**
1. Check all required fields are filled
2. Verify total amounts are reasonable
3. Try removing and re-adding items
4. Clear cart and start over if needed
5. Contact supervisor for assistance

### Customer Complaints
**For Order Issues:**
1. Listen to customer concern
2. Locate order in order history
3. Verify order details with customer
4. Contact manager for resolution authority
5. Document issue for follow-up

## Daily Operations Best Practices

### Efficiency Tips
- **Learn Hot Keys**: Memorize locations of popular items
- **Use Search**: Faster than browsing for known items
- **Batch Similar Orders**: Group similar orders when possible
- **Stay Organized**: Keep workspace clean and organized

### Accuracy Guidelines
- **Verify Orders**: Repeat order back to customer
- **Check Totals**: Ensure amounts are reasonable
- **Confirm Items**: Match items to customer request
- **Provide Receipts**: Always offer printed receipts

### Customer Service
- **Be Patient**: Allow customers time to decide
- **Ask Questions**: Clarify unclear requests
- **Suggest Alternatives**: For unavailable items
- **Stay Professional**: Maintain friendly, helpful attitude

### Cash Handling
- **Verify Amounts**: Double-check cash received
- **Provide Change**: Calculate change accurately
- **Secure Drawer**: Keep cash drawer closed when not in use
- **Report Discrepancies**: Note any cash handling issues

## Troubleshooting Common Issues

### Cart Problems
**Items Not Adding:**
- Refresh the page
- Check if item is marked as available
- Try searching for item by name
- Contact admin if item should be available

**Wrong Quantities:**
- Use +/- buttons to adjust
- Remove item completely and re-add
- Clear entire cart if needed

**Total Calculation Issues:**
- Verify GST rate is correct
- Check individual item prices
- Contact admin if calculations seem wrong

### Receipt Problems
**Won't Print:**
- Check printer power and connection
- Verify paper is loaded
- Allow pop-ups in browser
- Try print preview instead

**Missing Information:**
- Check if restaurant details are configured
- Contact admin to update receipt settings

### Performance Issues
**Slow Loading:**
- Check internet connection
- Close other browser tabs
- Refresh the page
- Clear browser cache if needed

**Freezing/Hanging:**
- Don't click repeatedly
- Wait 30 seconds for response
- Refresh page if no response
- Contact IT support
