# Admin Features Guide

This comprehensive guide covers all administrative features available to restaurant managers and administrators.

## Admin Dashboard Overview

### Accessing Admin Features
1. **Login**: Use Admin Login (not POS Login)
2. **Credentials**: Provided by super administrator
3. **Interface**: Additional tabs appear after admin login
4. **Permissions**: Full access to all system features

### Admin-Only Navigation
After admin login, you'll see additional tabs:
- **Menu**: Full menu management capabilities
- **Reports**: Detailed analytics and export features
- **Settings**: System configuration options
- **Orders**: Complete order history with export

## Menu Management

### Adding New Menu Items

#### Step-by-Step Process
1. **Navigate to Menu Tab**: Click "Menu" in main navigation
2. **Add New Item**: Click "Add New Item" button
3. **Fill Required Information**:
   - **Item Name**: Clear, descriptive name
   - **Price**: Include decimal points (e.g., 12.99)
   - **Category**: Select or create new category
   - **Image**: Upload item photo (optional)

#### Best Practices for Menu Items
- **Naming**: Use consistent naming conventions
- **Pricing**: Keep prices competitive and clear
- **Categories**: Group similar items logically
- **Images**: Use high-quality, appetizing photos
- **Descriptions**: Add detailed descriptions in notes if needed

### Editing Existing Items

#### Modification Process
1. **Find Item**: Browse menu or use search function
2. **Click Edit**: Use edit icon on item card
3. **Update Information**: Modify any field as needed
4. **Save Changes**: Click "Save" to update item

#### Common Edits
- **Price Updates**: Adjust for menu price changes
- **Availability**: Mark items as available/unavailable
- **Seasonal Changes**: Update items for seasons
- **Image Updates**: Replace with better photos

### Category Management

#### Creating Categories
- Categories are created automatically when adding items
- Use consistent category names across all items
- Consider logical grouping for customer ease

#### Recommended Categories
- **Appetizers**: Starters and small plates
- **Main Courses**: Primary dishes
- **Desserts**: Sweet items and treats
- **Beverages**: All drinks (alcoholic and non-alcoholic)
- **Specials**: Daily or seasonal specials

### Menu Organization Tips

#### Display Order
- Place popular items prominently
- Group similar items together
- Consider customer ordering patterns
- Update seasonal items regularly

#### Pricing Strategy
- Use psychological pricing (e.g., $9.99 vs $10.00)
- Maintain competitive pricing
- Consider profit margins
- Update regularly based on costs

### Saving Menu Changes

#### Important Notes
- **Save Requirement**: Always click "Save Menu to Server"
- **Confirmation**: Wait for success confirmation
- **Backup**: System creates automatic backups
- **Sync**: Changes appear immediately for all users

#### Troubleshooting Save Issues
- **Check Connection**: Ensure stable internet
- **Retry**: Click save again if first attempt fails
- **Validate Data**: Ensure all required fields are complete
- **Contact Support**: If persistent issues occur

## System Settings Configuration

### Restaurant Information

#### Basic Settings
- **Restaurant Name**: Appears on receipts and reports
- **Address**: Full business address
- **Phone Number**: Contact information for customers
- **Email**: Business email address

#### Receipt Configuration
- **GST Rate**: Tax percentage (e.g., 18.00 for 18%)
- **Currency Symbol**: Local currency symbol (₹, $, €, etc.)
- **Footer Text**: Custom message on receipts
- **Theme Color**: System color scheme

### Tax and Pricing Settings

#### GST Configuration
```
Default GST Rate: 18.00%
Applicable to: All menu items
Display: Separate line on receipts
Calculation: Automatic on all orders
```

#### Currency Settings
- **Symbol**: Choose appropriate currency symbol
- **Format**: Standard format (e.g., ₹12.99)
- **Display**: Consistent across all interfaces

- Custom thank you message

## Analytics and Reporting

### Daily Reports

#### Accessing Daily Analytics
1. **Navigate to Reports**: Click "Reports" tab
2. **Select Date**: Choose specific date for analysis
3. **Generate Report**: Click "Generate Daily Report"
4. **Review Data**: Analyze sales performance

#### Daily Report Contents
- **Total Sales**: Revenue for the selected day
- **Order Count**: Number of orders processed
- **Average Order Value**: Sales divided by order count
- **Popular Items**: Best-selling menu items
- **Hourly Breakdown**: Sales by hour of day

#### Using Daily Reports
- **Performance Tracking**: Monitor daily performance
- **Staff Evaluation**: Assess cashier performance
- **Inventory Planning**: Identify high-demand items
- **Scheduling**: Optimize staff scheduling

### Monthly Reports

#### Monthly Analytics Features
- **Total Monthly Revenue**: Complete month sales
- **Daily Breakdown**: Sales for each day
- **Trend Analysis**: Growth or decline patterns
- **Category Performance**: Sales by menu category

#### Strategic Uses
- **Business Planning**: Long-term strategy decisions
- **Menu Optimization**: Identify underperforming items
- **Marketing**: Plan promotional campaigns
- **Financial Planning**: Budget and forecast preparation

### Export and Data Management

#### Export Options
- **PDF Reports**: Professional formatted reports
- **CSV Data**: Spreadsheet-compatible data
- **Date Ranges**: Custom date range selection
- **Filtered Data**: Specific categories or items

#### Data Usage
- **Accounting**: Import into accounting software
- **Analysis**: Advanced analysis in Excel/Sheets
- **Archival**: Long-term record keeping
- **Compliance**: Tax and regulatory reporting

### POS Account Administration

#### Creating POS Accounts

1. **Account Planning**: Determine number of staff accounts needed
2. **Username Creation**: Use consistent naming convention
3. **Password Setup**: Assign secure initial passwords

#### Managing Existing Accounts
- **Password Resets**: Help staff with login issues

### Security Management

#### Password Policies
- **Minimum Length**: 8+ characters recommended
- **Complexity**: Mix of letters, numbers, symbols
- **Regular Updates**: Change passwords periodically
- **Unique Passwords**: Different for each account

#### Access Control
- **Role-Based**: Different access for POS vs Admin
- **Time-Based**: Restrict access to business hours if needed
- **Session Management**: Automatic logout after inactivity
- **Audit Trail**: Monitor login attempts and activities

## Troubleshooting Admin Issues

### Common Admin Problems

#### Cannot Access Admin Features
- **Login Type**: Ensure using Admin Login
- **Account Status**: Verify account is active
- **Role Verification**: Confirm admin role assignment
- **Browser Issues**: Clear cache, try different browser

#### Settings Not Saving
- **Network Issues**: Check internet connection
- **Permission Issues**: Verify admin permissions
- **Data Validation**: Ensure all required fields complete
- **Browser Storage**: Clear browser cache

#### Reports Not Generating
- **Date Range**: Verify date range is valid
- **Data Availability**: Ensure data exists for selected period
- **Permission Issues**: Confirm report access permissions
- **System Load**: Try during off-peak hours
